Typora 收费了

2021-12-04 · 2 min read
软件 Markdown

前言

一提到 Markdown 编辑器，相信大家都用过：VS Code、Atom、Sublime Text 3、VNote、Typora......对于我来说，VS Code 是地表最强编辑器，但不是专注于 Markdown 写作的；Atom 也牛逼，网络环境问题另说，我最讨厌强制安装在 C 盘的软件了😒，Sublime Text 无国内支付渠道，所以无法获得最佳体验；VNote 是开源的！但不支持实时渲染，而且 UI 界面我也不喜欢；Typora 是国人开发的，功能很多，最舒服的是它的实时渲染与打字机模式。

收费

虽然 Typora 的功能很多，体验很舒服，但也有一些毛病：在输入或打开一些大文件时，相对会卡顿不少，某一次我在写一份 5W 字数的文档时，莫名其妙发生了卡顿，然后没了，幸好有本地缓存文件可以恢复。😭😭😭
当时我还发邮件反馈了：

现在 Typora 都更新收费了，赶紧入正支持一下。毕竟也不贵，而且一个 License 可以同时在三个设备上使用（三个人一起平摊 $14.99，就是 ￥32），最重要的是一次性买断啊！！！

对于开发者来说，如果没有收入，也就没动力去频繁更新产品。

付费后，我在想：主题的创作者呢？主题是免费的，但是软件开始收费了，感觉有点被白嫖？后来在官网找到了回答：对于 Typora 有贡献者可以免费获得 License。

当然也可以使用以前的 beta 版本，或者改用其他的软件代替。

其他软件

Markdown 易读易写，使用群体广泛，笔记类应用又一直是豪杰辈出的领域，现在都基本都标配 Markdown 语法支持了。

推荐一些 Markdown 软件，个人版免费，当然开通会员更多功能✅。
