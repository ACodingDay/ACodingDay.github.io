实用软件推荐Ⅰ

2021-12-08 · 4 min read
Windows 推荐 软件

前言

“工欲善其事，必先利其器。”生活中，总需要一些实用软件来解决麻烦。我推荐一些在用的 Windows 软件。

💻 桌面壁纸软件

说起这个，想必大家的第一反应就是：Wallpaper Engine 与 upupoo！😍😍😍懂的都懂，不懂的也不必说了，但是下载的“妈见打”系列的壁纸，都不敢在公共场合展示啊。最近都出移动端了，冲！

而且，对于低配置的笔记本来说，没必要。所以，我日常使用的是腾讯电脑管家-桌面整理的独立版，没有广告没有捆绑，除了一键换壁纸（Unplash、Pexels 提供的壁纸），还有一键桌面整理，双击隐藏 / 显示桌面文件等功能。
缺点呢？说它是壁纸软件好像不合理，说它是整理软件好像不专业，但它二者兼有......😅

更节省内存的是换一张“四象限”的壁纸，参考这个知乎文章

📸 图片类软件

看图软件

Windows 自带的“照片”查看器支持的格式少，功能也较少，我现在使用的是免费开源的 ImageGlass。它支持的图片格式超过 72 种，预加载模式提高打开图片速度，从剪切板中打开图片，取色工具，Exif 工具，界面主题皮肤，中文语言支持......
如上图所示，图片的基本信息会显示在顶部工具栏的上方。

截图软件

Windows 自带的“截图和草图”功能也不强，一般来说，也可以使用微信、QQ自带的截图功能，但我不想为了截一张图去登录聊天工具啊。至于老牌的专业截图软件，例如：Snipaste 与 FastStone Capture，好用是好用，但是收费啊！😩
我现在用的是免费的 VeryCapture，除了常规的截屏功能（滚动长截屏），还有GIF 录制（帧率只有 10）、视频录制（帧率只有 20）、OCR、翻译......

如上图所示，截图后可以使用工具对图片进行标注、取色、OCR等编辑操作。

制图软件

这里不介绍什么 PS 与 Photoshop，介绍一个免费、有趣的制图软件：AndreaMosaic。
它的功能：让你将一张图片转换成一个由其他照片合集组成的镶嵌图，也就是所谓的蒙太奇效果。虽然功能很有趣，但是也不得不说，UI 界面啥的有亿点点务实（没有最小化、最大化、拖拽缩放）。

四个方案选择按钮，点击进去后也没有后退键，教程也是英语文档。如果详细的设置参数也有点令你头大，那么不妨看看这款工具：懒人图云。

用它来制作一些有纪念意义的图片再好不过了。

🎬 视频类软件

录制软件

说起专业又免费的录屏软件，那肯定是 OBS 了，它适配多平台、支持直播串流、高性能实时视频 / 音频捕获和混合，主播都在用。但我不需要这种高大上的专业功能。
我现在用的是 EV录屏，个人免费版的功能勉强够用了，但是一看它的VIP功能介绍，又感觉不够用......

因为录制 GIF 是 EV 会员才有的功能，我录制 GIF 使用的是免费开源的 ScreenToGif，主界面简约大方，安装包仅仅 3.6 MB（这位就是轻量级！！！），多国语言支持，主要用于对 GIF 帧进行编辑、修改、添加、删除，打开摄像头录制，画板编辑录制......

播放软件

首推多媒体播放器 PotPlayer 不解释，免费不流氓、内置各种解码器（硬、软解码），基本支持播放主流的视频 / 音频格式，快捷键支持，还可以变速播放、画声同步、实时字幕翻译、在线收看电视直播......
