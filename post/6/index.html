搜索引擎推荐Ⅰ

2022-01-12 · 5 min read
搜索引擎 推荐

前言

日常生活中，我经常会遇到一些难题，例如：看不懂的热词以及梗图、某领域的专业术语、谜语人的拼音缩写......这都让我十分头大，不得不去搜索了解。这时候就有必要用上称心如意的搜索引擎了。

推荐

当然，百度、必应、Google、Yandex、雅虎、DuckDuckGo、搜狗、360 等众所周知的就无需赘述。这里推荐的是一些少见的但用起来还不错的。

F 搜

官网地址：F 搜

主界面看上去简单干净，右上角有翻译、社区、工具集、注册登录等功能，搜索框下方有“搜索”和“实时”按钮，点击“实时”就会显示实时热度新闻，再次点击会隐藏；右下角有切换中文与英文搜索、帮助（使用说明书）、设置（搜索设置、主题模式、样式布局）等功能。它提供了无广告、过滤内容农场的搜索结果，支持翻译、天气、IP 查询、高级搜索等功能。

搜索结果的分类还不足够成熟，对比同类，它还可以增加一些“音频”、“视频”、“新闻”、“学术”等。右上角的狗头是处于实验中的功能（代码搜索、搜索结果排序算法），工具集中的工具基本都是给程序员用的。
搜索界面可以设置“居中显示”真的太爽了，我个人不喜欢左侧显示搜索结果，右侧大范围空白，因为在使用大屏幕时需要转头或者斜视。
注册登录后可以设置使用偏好、隐私与安全、同步信息等。

Petal Search

官网地址：华为花瓣搜索
华为提供的谷歌搜索替代品，现在 Petal Search 上线了网页版。因为主要是针对海外市场，所以提供了众多语言的搜索支持，可以设置显示语言、搜索语言、搜索区域等，内容安全搜索方面：支持设置三个等级、严格、中等、关闭，根据设定会过滤一些敏感的内容。
我也是从这篇文章：下1个好软件 得知的，但是我访问这个网页时，出现的是“403 Forbidden openresty”......😅😅😅
然后搜了一下，发现大家对于这个是自研的搜索引擎还是套壳的，争议还挺大的：知乎问答。

Searx

项目地址：searx
实例地址：SearXNG and SearX instances
这是一个基于 Python 的完全开源免费搜索引擎平台，提供来自 Google、Bing、Yahoo 等 70 多种各大视频、图片、搜索、磁力等网站结果展示，并对搜索结果进行优化，同时尊重隐私和允许二次开发。

这两个的可设置项大同小异，虽然是聚合搜索引擎，但也没必要点击“允许所有”，因为不是所有的搜索引擎都支持中文，有时候还不如本土的。

更多

还有一些不是很理想的搜索引擎。

  • 开发者搜索：百度的面向开发者的知识搜索平台，不是开发者就不需要。
  • 极客搜索：极客邦针对它全站内容资源的轻量级搜索引擎，内容来源极其有限。
  • 无追搜索：360最近新推出的号称简洁干净不追踪用户隐私的搜索引擎，截止目前 2022年1月12日，尚未适配手机版。
  • DogDogGo：主页的 logo 神似 DuckDuckGo 与多吉搜索，slogan 为“一个永无广告、永不收集个人信息的搜索引擎”。目前除了搜索，没有其他更多设置。
  • Magi：提供“时效性优先”和“信任度优先”排序搜索结果，但作者说了，本意不是做一个供日常使用的网页搜索引擎。

后续

其实没必要太纠结用哪一款搜索引擎，只要能过滤无用信息，得到想要的结果，用什么都没无所谓——哪怕用某度，也可以搭配油猴脚本去广告，去除特定内容。
至于如何高效使用搜索引擎，请参考大佬们的建议：知乎问答。
